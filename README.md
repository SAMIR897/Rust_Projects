<a href="https://github.com/SAMIR897/bulk_token_sender">
  <img src="./assets/bulk_token_sender.svg" alt="bulk_token_sender" width="760"/>
</a>

A high-performance Rust utility designed for efficient batch distribution of ERC-20 tokens and native assets. Enables developers to execute transactions to hundreds of recipients in a single call, significantly reducing gas costs and manual overhead. Built with safety and speed in mind — ideal for **airdrops** and **payroll distributions**.

<a href="https://github.com/SAMIR897/event_indexer">
  <img src="./assets/event_indexer.svg" alt="event_indexer" width="760"/>
</a>

A blazing-fast smart contract event indexer implemented in Rust. Monitors blockchain logs in real-time, parsing and storing live data into a structured format for low-latency querying. Essential for building responsive **dApps** that require immediate visibility into on-chain state changes and complex **event-driven business logic**.

<a href="https://github.com/SAMIR897/gas_tracker">
  <img src="./assets/gas_tracker.svg" alt="gas_tracker" width="760"/>
</a>

A sophisticated Ethereum gas tracking tool providing real-time insights into network congestion. Leveraging Rust's efficiency, it analyzes recent blocks to suggest optimal priority and base fees. Features **historical trend analysis** and customizable alerts — ensuring you never overpay or face unexpected delays during high network activity.

---

<a href="https://github.com/SAMIR897/mempool_monitor">
  <img src="./assets/mempool_monitor.svg" alt="mempool_monitor" width="760"/>
</a>

A specialized monitoring tool that peers into the Ethereum **txpool** to analyze pending transactions before they are mined. Captures critical data like gas prices and target contracts, providing a competitive edge for researchers. Its low-latency architecture ensures high-fidelity data capture — vital for real-time **MEV research** and network analysis.

<a href="https://github.com/SAMIR897/mini_chain">
  <img src="./assets/mini_chain.svg" alt="mini_chain" width="760"/>
</a>

A lightweight, educational blockchain implementation built from the ground up in Rust. Demonstrates core distributed ledger principles including cryptographic hashing, block linking, a **Proof-of-Work** consensus mechanism, and **peer-to-peer networking**. A perfect pedagogical resource for understanding how modern decentralized blockchains operate at a fundamental level.

---

<a href="https://github.com/SAMIR897/rpc_health_checker">
  <img src="./assets/rpc_health_checker.svg" alt="rpc_health_checker" width="760"/>
</a>

A robust monitoring utility designed to evaluate the performance and reliability of blockchain **RPC nodes**. Tracks latency, block height synchronization, and API availability across multiple networks simultaneously. Indispensable for infrastructure providers and dApp developers who demand high availability and consistency from their node providers.

---

<a href="https://github.com/SAMIR897/wallet_manager">
  <img src="./assets/wallet_manager.svg" alt="wallet_manager" width="760"/>
</a>

A secure CLI tool for generating and organizing hierarchical deterministic (**HD**) wallets using **BIP-39** and **BIP-44** standards. Written in Rust for maximum security — supports seed phrase generation, key derivation, and transaction signing. All cryptographic operations occur locally and securely, completely offline.

<a href="https://github.com/SAMIR897/impermanent_loss_calc">
  <img src="./assets/impermanent_loss_calc.svg" alt="impermanent_loss_calc" width="760"/>
</a>

An advanced DeFi calculator designed to help **Liquidity Providers** estimate potential impermanent loss and overall profitability within **AMMs**. By simulating price volatility and fee accrual, it provides clear risk-to-reward visualizations — enabling data-driven decisions when providing liquidity to decentralized exchanges and protocols.
